Rick and Morty Fan Page

This is a fan site for the show Rick and Morty, styled with Bootstrap v5. The webpage is not only dedicated to providing information about the characters, episodes, and locations of the show but also showcases the power of Bootstrap v5 in web design. All the information you find here is seamlessly pulled from the Rick and Morty API, demonstrating how technology can elevate your fan experience. Click on the links above to explore various pages, and find the links below to access the API and my Github profile.
